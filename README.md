# ระบบบริหารจัดการที่ดิน-LandManagement (LMAN)
### ระบบภูมิสารสนเทศเพื่อการบริหารจัดการทรัพย์สินที่ดินแบบครบวงจร

---

## คืออะไร

** ระบบบริหารจัดการที่ดิน(LMAN)** คือแพลตฟอร์ม GIS ที่รวมการดูแผนที่ ค้นหา วิเคราะห์เชิงพื้นที่ และออกรายงาน ไว้ในหน้าเว็บเดียว ไม่ต้องสลับใช้โปรแกรมแยกต่างหาก ช่วยให้การบริหารจัดการที่ดินและทรัพย์สินขององค์กรทำได้เร็วขึ้นและแม่นยำขึ้น

---

## ความสามารถหลัก

- 🗺️ **แผนที่ออนไลน์แบบโต้ตอบ** — ดูข้อมูล Vector และภาพถ่ายทางอากาศ พร้อมซ้อนข้อมูลจากหน่วยงานภายนอกได้
- 📐 **วิเคราะห์เชิงพื้นที่ในตัว** — Intersect, Union, Clip, Buffer และคำนวณพื้นที่/ระยะทางได้ทันทีบนเว็บ
- 📊 **Dashboard สรุปภาพรวม** — ข้อมูลเชื่อมโยงกันแบบเรียลไทม์ พร้อม Export เป็น PDF/Excel
- 🤖 **AI ผู้ช่วยอัจฉริยะ** — ถามข้อมูลด้วยภาษาธรรมดา AI จะดึงข้อมูล วิเคราะห์ และสรุปผลให้ทันที ไม่ต้องกรอก Filter หรือไล่ดูตารางเอง

---

## กระบวนการทำงาน

```
ค้นหา  →  แสดงผลบนแผนที่  →  วิเคราะห์เชิงพื้นที่  →  สรุปผล & รายงาน
```

---

## เหมาะกับใคร

หน่วยงาน/องค์กรที่ต้องบริหารจัดการที่ดินและทรัพย์สินจำนวนมาก และต้องการเชื่อมต่อข้อมูลกับหน่วยงานภายนอกผ่านมาตรฐาน OGC
